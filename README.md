# KaprekarsNumber
Little project to learn Python and calculate Kaprekar's numbers for any amount of digits.


Code sample: 
```python
>>> import kaprekar
>>> initialMapping = kaprekar.createFirstMapping(4)
>>> initialMapping.performMapping()
>>> printMapping(initialMapping)
```
